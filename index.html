How did I start developing websites and software?

Author: Dennis Loska, 11.12.2016

If you are new tro programming or web-development in general, you may ask yourself where to start. The best tip I can giver you is to just stick to the basics for a while, until you have mastered them all. If you wantr to become a frontend-developer for example, then master html, css and javascript first. You can worry about your workflow and which framework to use after that. If you want to challenge yourself - that´s great, but if the result is you quitting because of frustration then you should think about starting at a lower level, and raise the difficulty over time.
